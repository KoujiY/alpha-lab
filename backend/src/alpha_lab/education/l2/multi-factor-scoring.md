# 為什麼要多因子

單一因子（例如只看 PE）在不同市場環境表現不穩：牛市追高者拿不到低 PE、熊市價值陷阱層出不窮。學術與實務（Fama-French 等）早已證明：把**低相關**的因子組合，長期報酬更穩。

alpha-lab 採用四因子：

| 因子 | 代表指標 | 偏好 |
|------|---------|------|
| Value（價值） | PE、PB | 便宜 |
| Growth（成長） | 營收 YoY、EPS YoY | 長得快 |
| Dividend（股息） | 殖利率（未來加配息穩定） | 配得起 |
| Quality（品質） | ROE、毛利率、負債比、FCF | 體質好 |

## 橫截面百分位法

每個因子的指標單位不同（PE 是倍數、殖利率是百分比），需要**歸一化**才能加起來。本系統用「橫截面百分位」：

> 同一個計算日（`calc_date`），把當天全體台股 A 指標排名，每檔在排名中的百分位就是它的分數（0-100）。

好處：
- 不用 pick 絕對閾值（PE 多少算低？爭論無解）
- 自動隨市場調整（熊市整體 PE 都低，百分位分布依然穩）
- 同指標 None 的標的不會被計入

## 從因子分到總分

1. 計算四個因子各自的 0-100 分
2. 依**風格權重**加權：
   - 保守組：`0.20 Value + 0.10 Growth + 0.35 Dividend + 0.35 Quality`
   - 平衡組：`0.25 × 4`
   - 積極組：`0.15 Value + 0.50 Growth + 0.05 Dividend + 0.30 Quality`
3. None 因子自動被略過，剩餘權重 renormalize

## 從總分到組合

1. 依總分排序取 Top 30
2. 同產業最多 5 檔（避免過度集中）
3. 最終取前 10 檔
4. 權重 = `softmax(總分 / 20)`，單檔上限 30%

## 為什麼這樣設計

- **百分位**穩健、自動適應市場
- **風格權重**讓使用者表達偏好
- **產業分散 + 單檔上限**控制單一事件風險
- **softmax**讓高分檔領到顯著高權重，但不會 winner-takes-all

## 局限

- 只看歷史財報與價格，未涵蓋新聞、事件、宏觀面
- 產業分類來自 TWSE，粗糙（半導體一類裡其實差異很大）
- 四因子沒有「動量」因子（技術面）
- 推薦**不等於**買進訊號，使用者要自己做最後判斷
